# Hair by Gashi

Eksklusiv frisørsalon i Aalborg, Danmark.

## Om projektet

Dette er den officielle hjemmeside for Hair by Gashi - en luksus frisør- og barbersalon i hjertet af Aalborg.

## Teknologier

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Lokal udvikling

```sh
# Klon repository
git clone <YOUR_GIT_URL>

# Naviger til projektmappen
cd <YOUR_PROJECT_NAME>

# Installer dependencies
npm install

# Start udviklingsserver
npm run dev
```
